# TOP500 算力数据可行性报告

**日期**: 2026-07-04
**数据源**: top500.org XML (1993-2025, 65 期, 每期 500 系统)
**产出文件**: `data/interim/compute_capacity_annual.parquet`, `data/interim/growth_with_compute.parquet`

---

## 数据获取

| 项目 | 数值 |
|---|---|
| TOP500 原始系统记录 | 32,500 条 |
| 覆盖期数 | 65 期 (1993-06 ~ 2025-06) |
| 原始国家数 | 65 |
| ISO3 映射后国家数 | 57 |
| 未映射实体 | Venezuela, Puerto Rico, Liechtenstein, Belarus, Saudi Arabia (拼写变体) |

## 算力变量

| 变量 | 含义 | 单位 |
|---|---|---|
| `log_total_rmax` | log(国家总算力), 0 若无 TOP500 系统 | log(GFlops) |
| `n_top500_systems` | 该国在 TOP500 中的系统数 | 个 |
| `total_rmax_pflops` | 该国总算力 | PFlops |
| `total_cores` | 该国总核心数 | 个 |
| `best_rank` | 该国最佳排名 | 排名 |

聚合方式：每期 (6月/11月) 各占 500 系统 → 取半年均值 → 年度均值。

## 关键发现

### 全球算力趋势

全球总算力从 2015 年的 391 PFlops 增长到 2023 年的 6,137 PFlops，8 年增长 **15.7 倍**。

| 年份 | 全球总算力 (PFlops) |
|---|---|
| 2015 | 391 |
| 2016 | 620 |
| 2017 | 799 |
| 2018 | 1,313 |
| 2019 | 1,612 |
| 2020 | 2,319 |
| 2021 | 2,912 |
| 2022 | 4,639 |
| 2023 | 6,137 |

### 覆盖率

- 2015+ 总面板：1,665 obs (185 国)
- 有 TOP500 系统：282 obs (44 国)
- **算力观测占比：16.9%**；**有算力国家占比：23.8%**

### 算力集中度

美国占总算力的约 50%（2023 年 3,063 PFlops），中美日三国占约 68%。

### 简单关联

- corr(growth_annual, log_total_rmax) = 0.024
- 有算力国家 mean growth = 1.61%
- 无算力国家 mean growth = 1.23%

> ⚠️ 这些相关不可解释为因果。算力与收入水平高度相关，需控制固定效应和滞后控制变量。

## 可行性判断

| 维度 | 评估 |
|---|---|
| 数据可获得性 | ✅ 65 期 XML 已全部下载缓存，可复现 |
| 国家覆盖 | ⚠️ 57 国有数据，但 2015+ 仅 44 国。大部分发展中国家无 TOP500 系统 |
| 变量操作化 | ✅ 已有 `log_total_rmax`、`n_top500_systems` 两个互补代理 |
| 与 PWT 对接 | ✅ 已 merge 到 `data/interim/growth_with_compute.parquet` |
| 识别可行性 | ⚠️ 算力明显内生。FE 回归仅消除不随时间变化的混淆，但时变混淆（如经济增长→算力投资的反向因果）仍需外生冲击 |

**结论：R1 数据路径可行，但需注意**：
1. 算力变量在大约 141 个发展中国家取 0（无 TOP500 系统），这既是变量特征也是识别困难
2. 建议将样本限制在有至少一次 TOP500 出现的国家（44 国）做主要分析，零算力国家做辅助分析
3. OEC GPU 进口数据可作为补充代理，覆盖可能更广（后续 step）
