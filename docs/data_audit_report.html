6. 已知数据特性

  • PWT 进口份额 (pwt_import_share) 全部为负值——PWT 标准约定 csh_m = −(M/GDP)。回归中取绝对值或与 pwt_export_share 相加得到贸易开放度。
  • 极端增长值：56 个观测 |growth_annual| > 0.30，集中在战乱/石油国家。稳健性检查应排除。
  • TOP500 算力变量：仅 data/interim/growth_with_compute.parquet 包含，growth_panel_1960plus 不含。

⚠️ 未下载的 WDI 指标

  • SP.URB.TOTL.IN.ZS (wdi_urban_population_pct)
  • SP.DYN.LE00.IN (wdi_life_expectancy)

网络超时导致无法从 World Bank API 获取这 2 个指标。build_panels.py 已做容错，pipeline 不因此崩溃。
